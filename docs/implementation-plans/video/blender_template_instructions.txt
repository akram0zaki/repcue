
# Blender Scene Template Instructions for RepCue Exercise Clips

This template ensures consistent lighting, camera, and world setup for all exercise animations.

1. **Scene Units & World**
   - Start a new Blender scene (File > New > General).
   - Delete default cube, light, camera.
   - Go to Scene Properties > Units: set Unit Scale = 1.0, Length = Metric.
   - In World Properties, enable 'Use Nodes' and set Background color to mid-gray: RGBA = (0.1, 0.1, 0.1, 1.0) or Hex #1a1a1aff, Strength = 1.0.

2. **Ground Plane**
   - Add > Mesh > Plane, scale to 20 m.
   - Create a new Material "GroundMat", Principled BSDF Base Color = (0.12, 0.12, 0.12, 1.0). Assign to plane.

3. **Camera Setup**
   - Add > Camera. 
   - In Object Properties: 
     - For standing moves: Location = (0.0, -4.2, 1.5).
     - For floor moves: Location = (0.0, -3.2, 1.0).
     - Rotation: X = 80°, Y = 0°, Z = 0° (slight down tilt).
   - Camera Properties: Focal Length = 50 mm, Sensor Fit = Auto.

4. **Lighting**
   - Add > Light > Area: Name = "Key", Location = (3, -3, 3), Rotation = (-35°, 0°, 35°), Power = 1500W, Size = 5 m.
   - Add > Light > Area: Name = "Fill", Location = (-3, -3, 2.5), Rotation = (-25°, 0°, -35°), Power = 800W, Size = 5 m.
   - Add > Light > Area: Name = "Rim", Location = (0, 3, 2.5), Rotation = (-160°, 0°, 0°), Power = 600W, Size = 3 m.

5. **Importing Animations**
   - File > Import > FBX (or GLB). Use 'Automatic Bone Orientation' for FBX.
   - Ensure armature and mesh are centered at origin, feet on ground plane.

6. **Loop Timing**
   - In the Action Editor, set frame_start = 1, frame_end so that the cycle is exactly one rep or desired hold duration.
   - For seamless loops, ensure first/last frames match or use 'Cycle' modifier on f-curves.

7. **Static Pose Idle**
   - For holds (e.g., plank), add tiny sway: select armature root bone, insert rotation keyframes at frame 1 and end, with ~2° variation on Z or X.

8. **Render Settings**
   - Output Properties:
     - Resolution presets: 1080×1080, 1080×1920, 1920×1080 (switch as needed).
     - Frame Rate: 30 fps.
     - File Format: FFmpeg Video, Encoding: MPEG-4, Codec: H.264, Quality: Perceptually Lossless.
   - Output file path pattern: /videos/{exercise_id}_v1_{WxH}.mp4

9. **Saving Template**
   - Save as 'repcue_template.blend'. For each exercise, open template, swap animation, adjust camera height if floor vs standing, set correct aspect ratio, render.
